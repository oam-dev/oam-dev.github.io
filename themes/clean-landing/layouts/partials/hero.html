Star

An open standard for defining cloud native apps.

Open Application Model [OAM] is runtime-agnostic model that defines cloud native apps by natural and enables building platforms where developers focus on their apps only.

With consistent abstractions to declare operational features of the app, and standard interfaces to build platform capabilities in app-centric approach.

Open Application Model is highly extensible.
